Бұл нұсқаның ерекшеліктері

Акселератор күні бета-тестілеу мақсаттары үшін осы нұсқаға енгізілген.

Дерекқор көшірмелерінің механизмі бета-тестілеу мақсаттары үшін осы нұсқаға енгізілген.

Дербес сервер бета-тестілеу мақсатында осы нұсқаға қосылған.

Лицензиялық келісім

Лицензиялық келісім, 1С:Кәсіпорын 8 пайдалану ережелерін реттейтін файлда келтірілген  1CEnterpise_ru.htm.

Алдыңғы нұсқадан өту

1С:Кәсіпорын 8.3 тәуелсіз компьютерге орнатылады 1С:Кәсіпорын 8.0, 1С:Кәсіпорын 8.1, 1С:Кәсіпорын 8.2 және бір мезгілде пайдаланылуы мүмкін осы нұсқалармен  клиенттік компьютерде де, серверде де.

Ағымдағы нұсқаның айырмашылықтары алдыңғы нұсқалардан және жаңа нұсқаға өту ерекшеліктері файлда келтірілген V8Update.htm.
