Особливості цієї версії

Дата акселератор включений у поточну версію для цілей бета-тестування.

Механізм копій бази даних включений у поточну версію для цілей бета-тестування.

Автономний сервер включений у поточну версію для цілей бета-тестування.

Користувацька угода

Користувацька угода, що регулює правила використання 1С:Підприємство 8, наведена у файлі  1CEnterprise_ru.htm.

Перехід з попередніх версій

1С:Підприємство 8.3 встановлюється на комп'ютер незалежно від 1С:Підприємства 8.0, 1С:Підприємства 8.1, 1С:Підприємства 8.2 і може використовуватися одночасно з цими версіями як на клієнтському комп'ютері, так і на сервері.

Відмінності поточної версії від попередніх версій та особливості переходу на нову версію приведені у файлі V8Update.htm.
