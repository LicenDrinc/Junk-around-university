Propiedades de esta versión

El cliente móvil está incluido en esta versión con el objetivo de beta-test.

El mecanismo del sistema de interoperabilidad y el servidor de interoperabilidad están incluidos en esta versión con el objetivo de beta-test.

El cliente de 64 bits para Windows está incluido en esta versión con el objetivo de beta-test.

El mecanismo del uso del configurador en el modo de agente está incluido en esta versión con el objetivo de beta-test.

El mecanismo de actualización de la configuración de la base de datos optimizado está incluido en esta versión con el objetivo de beta-test.

El soporte de OS Windows en la plataforma móvil está incluido en esta versión con el objetivo de beta-test.

Acuerdo de licencia

El acuerdo de licencia que controla las reglas del uso de 1C:Empresa 8 se encuentra en el archivo 1CEnterpise_en.htm.

Traspaso de las versiones anteriores

1C:Empresa 8.3 se instala en el ordenador independientemente de 1C:Empresa 8.0, 1C:Empresa 8.1, 1C:Empresa 8.2 y puede ser usada simultáneamente con estas versiones como en el ordenador de cliente tanto en el servidor.

Las diferencias de la versión actual de las versiones anteriores y peculiaridades de traspaso a la versión nueva se encuentran en el archivo V8Update.htm.
