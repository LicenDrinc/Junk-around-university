Caractéristiques de cette version

Un accélérateur des données est inclus dans cette version à des fins de test bêta.

Le serveur autonome est inclus dans cette version à des fins de test bêta.

Contrat de licence

Le contrat de licence régissant l’utilisation de 1C:Enterprise 8 est indiqué dans le fichier 1CEnterprise_en.htm.

Transition des versions précédentes

1C:Enterprise 8.3 est installé sur un ordinateur indépendamment de 1C:Enterprise 8.0, 1C:Enterprise 8.1, 1C:Enterprise 8.2 et peut être utilisé simultanément avec ces versions à la fois sur l'ordinateur client et sur le serveur.

Les différences entre la version actuelle et les versions précédentes et les fonctionnalités de la transition vers la nouvelle version sont indiquées dans le fichier V8Update.htm.
