Besonderheiten dieser Version

Datumsbeschleuniger ist in dieser Version für Beta-Testzwecke enthalten.

Der Mechanismus der Datenbankkopien ist in dieser Version für Beta-Testzwecke enthalten.

In dieser Version ist ein eigenständiger Server für Beta-Testzwecke enthalten.

Lizenzvereinbarung

Die Lizenzvereinbarung, die die Verwendung von 1C: Enterprise 8 regelt, ist in der Datei 1CEnterprise_en.htm enthalten.

Übergang von früheren Versionen

1C: Enterprise 8.3 wird unabhängig von 1C: Enterprise 8.0, 1C: Enterprise 8.1, 1C: Enterprise 8.2 auf einem Computer installiert und kann gleichzeitig mit diesen Versionen sowohl auf dem Client-Computer als auch auf dem Server verwendet werden.

Die Unterschiede der aktuellen Version zu den Vorgängerversionen und die Funktionen des Übergangs zur neuen Version sind in der Datei V8Update.htm angegeben.
